# Testimonial Cards

**Project URL:** https://roadmap.sh/projects/testimonial-cards

A responsive testimonial cards component built with HTML and CSS, demonstrating various layout and positioning techniques including Flexbox, Grid, absolute positioning, and pseudo-elements.

## Features

- 4 different testimonial card layouts
- Responsive design
- Pure HTML & CSS (no frameworks)
- Star ratings with CSS clip-path
- Speech bubble with CSS triangle pointer

## Layouts

1. **Dark speech bubble** — Koyu arka planlı konuşma balonu, `::after` ile üçgen ok, `position: absolute`
2. **Horizontal with avatar** — `display: flex` ile yatay düzen, solda büyük avatar, sağda içerik
3. **Image overlay card** — `position: absolute` ile resim üzerine metin yerleştirme
4. **Dark simple card** — Koyu arka planlı, sade ve okunabilir kart
